CS 134 Computer and Network Security - Gene Tsudik


内容:Computer and Network Security

Professor: Gene Tsudik

难易程度: ★★★★

先修科目: ICS 6D & ICS 33/CSE 43 & CS 132/CS 143A

TA:Ivan Nunes, Ercan Ozturk

Discussion: 无

Piazza:无

Textbook:Network Security - Private Communication in a Public World

Ratemyprofessor: 4.1

Grading Criteria:

· Homeworw 拿85%以上很难，TA评分吝啬

· Quiz

· Final 简单一点，有curve

Comment:教授比较幽默，讲课还是不错的，语速属于pattis等级，说话也很清晰。一般会迟到5分钟左右。"可谓GPA杀手，当然对于学霸来说也许也没什么，如果很珍惜自己的GPA，或者没有信息能完全掌握要求的内容，或者打算通过minimum的努力和机智拿A的同学，还是尽量不要作死，但这些纯属个人意见，大家可以去试听，当然也有一个风险是听完觉得没什么，然后第一次作业或者midterm成绩出来发现爆炸了。其中RSA等加密算法会比6D讲得深入一些，学的数学也稍微难一点，对modulo生疏的同学要注重研究一下（当然不是指计算remainder...），但这些毕竟只是这节课的一个部分，大部分还是非数学的，MIDTERM前相对math heavy，midterm后就没有了，protocol为主，主要是对一些protocol和攻击方法的理解。

另外也许有些人以为会教你怎么去hack别人，no。但你能学到一些原理，具体的自己去探索了。

建议感兴趣的同学，拿这节课的syllabus和textbook看一下，但是network方向的同学反正不能躲，加油吧！如果对network感兴趣的话，其实这是一门很有意思的课，也能学到不少网络安全知识，认真学，作业保持85以上，midterm不要崩，B+以上也许没问题？po主渣渣做不到但不代表学霸们不行。"
